# Mosaic-XR-Unity

## Environment

Equipment: Meta Quest 3 (OS Version: v81.0)
Unity Version: 6000.0.53f1

## Instructions

安裝 Unity Hub 及對應之 Unity 版本（6000.0.53f1）

開啟後按照伺服器環境修改連線目標 IP 及 Port
![p1](public/p1.png)

請確保在 Meta Quest 上啟用 USB 偵錯（adb）、開啟開發人員設定（進階 -> 啟用開發人員設定）、允許 App 存取頭戴式裝置相機權限。
